Principais Formatações

Negrito/Destaque

Nesta frase temos um negrito usando a tag <b> (Não semântica)

Nesta frase temos um termo em destaque usando a tag <strong> (semântica de destaque)

Itálico/Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (Não semântico)

Nesta frase temos um termo em ênfase usando a tag <em> (Semântica de ênfase)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto Grande e Pequeno

Estamos criando um texto GRANDE e um texto pequeno usando as tags <big>(OBSOLETA) e <small>

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser desconsiderado na hora da leitura usando a tag <del>

Texto Inserido

Podemos marcar um texto como inserido para indicar que foi adicionado depois usando a tag <ins>

Texto Sobrescrito

Para inserir x2+3 com a tag <sup>

Texto Subescrito

Para inserir coisas do tipo H2O com a tag <sub>
